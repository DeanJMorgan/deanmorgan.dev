# British English Standards

This skill ensures all written content on the site adheres to British English conventions, maintaining a consistent brand voice for Dean Morgan's UK-based consultancy.

## Core Principles

- **Spelling**: Use '-ise' instead of '-ize' (e.g., _optimise_, _modernise_, _specialise_).
- **Vocabulary**: Use British terms (e.g., _bespoke_ instead of _custom_, _programme_ instead of _program_, unless referring to computer code).
- **Tone**: Maintain a professional, authoritative, yet approachable tone common in British technical consultancy.

## Prohibited Patterns

- DO NOT use Americanised spelling in UI labels, meta descriptions, or blog content.
- DO NOT use 'color' in user-facing text; use 'colour'.
- DO NOT use 'center' for layout descriptions in text; use 'centre'.

## Verification

- When generating content, double-check all words ending in 'ize/ise' or containing 'or/our'.
- Use the `grep_search` tool to audit existing files for Americanisms if requested.
